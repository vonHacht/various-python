# shortest path

> **shortest path**\
> d(A, D) = 10 (**A**-->**C**-->**B**-->**D**)\
> d(E, B) = inf

> **weight of edge**\
> w(A, D) = 20
> w(A, B) = 10

> **shortest path from S to X**\
> d(S, X) = min( d(S, U) + w(U, X) )
